# 📱 Simple Calculator App (Android - Java)

A clean and simple calculator app built using **Java in Android Studio**.
This project demonstrates basic Android UI design, button click handling, and custom expression evaluation logic.

---

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- 🔄 Clear (C) button
- 🎯 Real-time button input
- 🧮 Custom expression evaluator (No external libraries)

---

## 🛠 Built With

- **Java**
- **Android Studio**
- **XML Layouts**
- **AppCompatButton**
- **Custom Math Parser**
